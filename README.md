Coffee Sales Dashboard – Excel Data Analytics Project

An interactive Coffee Sales Dashboard built in Microsoft Excel to analyze sales performance, customer insights, and product trends using dynamic charts, slicers, and pivot tables.

##Dataset used:

-<a herf="https://github.com/Buvanesh-R/Excel-projects/blob/main/coffeeOrdersDataDashboard.xlsx">Dataset used</a>

📌Project Overview:
This dashboard helps visualize coffee sales data across different countries, roast types, package sizes, and customer segments. It provides clear business insights through interactive filtering and graphical analysis.

🚀Features:
-Dynamic timeline filter for sales analysis by year and month
-Sales distribution by country
-Top 5 customers analysis
-Roast type filtering (Dark, Light, Medium)
-Package size analysis (0.2 Kg, 0.5 Kg, 1.0 Kg, 2.5 Kg)
-Loyalty card customer segmentation
-Interactive line chart for total sales over time
-Clean and professional dashboard design

🛠Tools & Technologies Used:
-Microsoft Excel
-Pivot Tables
-Pivot Charts
-Slicers & Timeline Filters
-Data Cleaning & Formatting
-Dashboard Design

📊Insights Generated:
-Identified top-performing countries in coffee sales
-Analyzed customer purchasing behavior
-Compared sales trends across roast types and product sizes
-Tracked monthly and yearly sales growth
